hexo初体验

发表于 2017-07-30

前提

配置很简单，为此我们需要安装三个软件，即Git Node.js hexo

一个Github的账号。之前配置不成功是因为下载的Node.js版本太老，去官网下载了最新的版本才能成功安装hexo,建议安装Node.js可以用nvm的方式，可以参考http://wiki.jikexueyuan.com/project/hexo-document/

主题

安装Git

跟着一步一步安装，还有需要配置.ssh

https://jingyan.baidu.com/article/90895e0fb3495f64ed6b0b50.html

安装Node.js

一直下一步即可

安装hexo

重点来了,我们需要新建一个文件夹blog,在该目录下打开命令窗口或者使用Git Bash

$ npm install -g hexo-cli

依旧在该目录下输入命令

$ hexo init       
$ cd blog
$ npm install

到此差不多快完成了,使用hexo s -g就可以启动hexo，打开浏览器，输入localhost:4000,就可以在本地看到你的个人博客了。

gitHub

这里我们需要去new一个仓库，并将该仓库page改为自己的域名，如果没有域名就通过yourname.github.io访问

安装hexo git插件

$ cnpm install hexo-deployer-git --save

发布更新博客

$ hexo d -g
发布成功后即可使用yourname.github.io访问你的网站了
